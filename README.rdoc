= orangewire_sender

Sends messages to The Orangewire

== Usage

Set OrangewireSender.host_url to the url for your Orangewire console.

    OrangewireSender.host_url = "http://localhost:3000"
    
Call 

    OrangewireSender.notify("headline", "summary text")
